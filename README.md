# Mist
An minecraft server implementation in rust for minecraft 1.21.10

## Features
**General**
- [x] Config (.toml)

**Networking**
- [x] Inital handshake
- [x] Server info querying
- [x] Encryption
- [x] Mojang authentication
- [x] Enter play state

**World**
- [ ] Show players in tablist
- [x] Spawning player
- [ ] Player skins
- [x] Sending chunks
- [x] Fullbright lightning
- [ ] Proper lightning
- [x] Flat worldgen
- [ ] Improved flat worldgen (ores and stone/deepslate from -64 to 64)
- [ ] Proper worldgen
- [x] World saves
